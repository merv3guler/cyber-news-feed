An automated, serverless Threat Intelligence Aggregator that collects, analyzes, and summarizes the latest cybersecurity news using Artificial Intelligence.

## ğŸš€ How It Works

This project leverages **GitHub Actions** to run a fully automated pipeline every day. No servers, no databases, just pure code.

1.  **Collection:** A Python script scrapes RSS feeds from trusted sources like *The Hacker News*, *BleepingComputer*, and *CISA*.
2.  **Analysis:** Each article is analyzed by **Google Gemini 1.5 Flash AI** to determine if it's a "Zero-Day" threat.
3.  **Summarization:** The AI generates a technical, one-paragraph summary in English.
4.  **Archiving:** New articles are merged with the historical data (`articles.json`) to keep a persistent record.
5.  **Deployment:** The results are rendered into a Cyberpunk-themed HTML dashboard and deployed to **GitHub Pages**.
